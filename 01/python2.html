Введение

На сайте https://docs.python.org/3/ размещена официальная документация (на английском языке) к последней стабильной версии языка Python 3.

Официального перевода на русский язык не делается. Неофициальные версии перевода можно найти в Сети, но следует знать, что

  • перевод документации к последней версии языка обычно отсутствует;
  • как и многие иные подобные тексты на русском языке переводы официальной документации часто содержат опечатки и даже грубые ошибки.

При этом отсутствие качественной русскоязычной документации не является реальной проблемой, так как работа с языком Python возможна и в случае полного незнания английского языка. А уже совсем небольших знаний английского достаточно для чтения документации (к тому же есть и Google Переводчик).

The Python Tutorial

Сегодня мы познакомимся с разделами 3 и 4 руководства, которые описывают важнейшие типы данных языка Python и его базовые управляющие конструкции. Приведённые в этих разделах примеры кода иллюстрируют использование Python-интерпретатора. В случае размещения программ в файлах применение описываемых конструкций не меняется.

Числа (numbers)

Прежде всего нас интересуют типы int, float и complex. Посмотрите внимательно на примеры, приведённые в секции 3.1.1.

Обратите особое внимание на операции /, // и %.

Строки (strings)

Строкам посвящена секция 3.1.2.

Здесь надо обратить внимание на операции конкатенации (+), повторения (*) и, особенно, вырезки (slicing), а также функцию len.

Вырезка [start:stop:step] имеет следующие важные свойства:

  • положительные индексы начинаются с нуля (самый левый символ строки);
  • отрицательные индексы «начинаются» с минус единицы (самый правый символ);
  • start всегда включается в вырезку, stop никогда не включается;
  • отсутствующий start считается нулевым;
  • отсутствующий stop считается равным размеру строки;
  • отсутствующий step считается равным единице.

Вот некоторые примеры:

>>> s = "0123456789"
>>> s[1]
'1'
>>> s[-2]
'8'
>>> s[1:4]
'123'
>>> s[::-1]
'9876543210'
>>> s[2:5:-1]
''
>>> s[5:2:-1]
'543'
>>> s[-2:-6:-2]
'86'

Списки (lists)

Списки в языке Python являются своеобразными аналогами массивов в некоторых иных языках программирования.

В отличие от неизменяемых (immutable) строк списки представляют собой изменяемый (mutable) тип данных: элементы списка можно изменять, их можно удалять и добавлять новые элементы в произвольную позицию.

К спискам также применимы операции конкатенации, повторения и вырезки. С помощью присваивания вырезке списка можно:

  • вставить в него другой список:

    >>> a = [1,2,3]
>>> a[1:1] = [5,6]
>>> a
[1, 5, 6, 2, 3]

  • удалить из него подсписок:

    >>> a = [1,2,3,4,5]
>>> a[1:3] = []
>>> a
[1, 4, 5]

  • заменить подсписок на другой список:

    >>> a = [1,2,3,4,5]
>>> a[1:4] = [7,8,9]
>>> a
[1, 7, 8, 9, 5]

Функция range и управляющие конструкции for, break и continue

Секция 4.2 иллюстрирует также очень активно используемую конструкцию for (своеобразный цикл). Очень часто она используется в сочетании с функцией range, возвращающей диапазон (фрагмент арифметической прогрессии) чисел.

Секция 4.3 содержит описание очень активно используемой функции range, которая в общем случае имеет вид range(start, stop, step). Обратите внимание на тот факт, что для получения списка кроме вызова функции range необходим ещё и вызов функции list.

В секции 4.4 описаны конструкции break и continue, позволяющие изменять нормальный ход выполнения циклов. Заметьте, что конструкцию else для цикла for использовать не рекомендуется.

Простейший случай определения функции

Функции полезны для устранения дублирования кода. При этом функции в языке Python — достаточно сложные объекты. В данный момент мы собираемся познакомиться лишь с самым простым вариантом определения функции, речь о котором идёт в секции 4.6.

Пока нам нужно лишь уметь определять функции с фиксированным числом аргументов с помощью ключевого слова def и вызывать их.
