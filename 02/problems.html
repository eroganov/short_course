Минимум функции

Найдём минимальное значение функции y=x2−3∗x+1 на множестве целочисленных точек отрезка [−5,4]:

print(min(map(lambda x: x * x - 3 * x + 1, range(-5, 5))))

Эта программа иллюстрирует использование анонимных (или лямбда) функций и встроенных (built-in) функций map и min.

«Интересные» четырёхзначные числа

Вот так можно найти минимальное четырёхзначное число, которое при перестановке его последней цифры на первую позицию уменьшается на 27:

print(next(filter(
  lambda x: x // 10 + 1000 * (x % 10) == x - 27, range(1000, 10000))))

С дополнительной информацией про итератор filter, который в языке Python тоже является встроенной функцией, можно познакомиться в официальной документации.

Все четырёхзначные числа, обладающие указанным свойством, можно напечатать так:

for n in filter(
  lambda x: x // 10 + 1000 * (x % 10) == x - 27, range(1000, 10000)):
    print(n)

Палиндром

Если программный код, приведённый на следующем слайде, разместить в файле с именем palindrome.py и сделать этот файл выполняемым с помощью команды chmod +x palindrome.py (в операционной системе Linux), то запускать программу будет очень удобно:

[roganov@aorus tmp]$ ./palindrome.py рокот
False
[roganov@aorus tmp]$ ./palindrome.py топот
True
[roganov@aorus tmp]$ ./palindrome.py а роза упала на лапу азора
True
[roganov@aorus tmp]$
#!/usr/bin/env python3
import sys


arg = "".join(sys.argv[1::])
print(arg == arg[::-1])

Этот код показывает, как просто в языке Python инвертировать строку (список инвертируется таким же образом) и работать с аргументами командной строки.

Проверка на простоту

Вот один из не самых плохих (хотя и далеко не самый эффективный) способов проверки целого числа, большего единицы, на простоту:

from math import sqrt


def is_prime(n):
    return n == 2 or (n % 2 != 0 and all(n % i != 0 for i in range(
        3, int(sqrt(n) + 1), 2)))


print(is_prime(int(input("n -> "))))

Этот код иллюстрирует использование встроенной функции all и языковой конструкции list comprehension. На русский язык этот термин переводят то как списковое включение, то как представление списка, то как генератор списков. Последний термин наименее удачен, так как в языке Python есть понятие генератора и это вовсе не list comprehension.

Счастливые билеты

from functools import reduce


def s(n, i):
    if n == 1:
        return 1 if (0 <= i and i < 10) else 0
    else:
        return reduce(lambda t, j: t + s(n - 1, i - j), range(10), 0)


def happy(n):
    return reduce(lambda t, j: t + s(n, j)**2, range(9 * n + 1), 0)


print(happy(int(input("Половина количества цифр в билете-> "))))
